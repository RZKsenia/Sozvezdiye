## 📖 Описание
Система «Созвездие» — это одностраничное веб-приложение (SPA), имитирующее рабочую среду сотрудника НИИ «Предельные Энергии» на острове Иов. Приложение предоставляет оператору инструменты для мониторинга резервуаров с тёмной материей, анализа аномальных событий, управления реестром сгустков и визуализации энергетических потоков. Интерфейс выполнен в стилистике научного терминала с учётом атмосферы романа: тёмная тема, scanlines, неоновые индикаторы и моноширинная типографика.

## 🚀 Ключевые возможности
- 📊 **Единый рабочий экран**: Дашборд с метриками и интерактивная мнемосхема отображаются одновременно.
- 🗺️ **Мнемосхема хранилища**:
  - Плавное панорамирование (`drag`) и масштабирование (`scroll`)
  - Анимация динамического уровня заполнения резервуаров
  - Кликабельные элементы с выводом детальной карточки резервуара
- 📜 **Журнал событий**: Двухпанельный интерфейс (список слева + детали и график справа) без модальных окон.
- 📦 **Реестр сгустков**: Классификация по типу духа, фильтрация, кнопки управления (`В резервуар` / `Переработка`) с обновлением дашборда в реальном времени.
- 🎨 **Атмосферный UI**: CSS-переменные, glassmorphism, SVG-графика, адаптивные анимации и эффект "монитора терминала".

## 🛠 Технологический стек
- **Frontend**: Vanilla HTML5, CSS3 (Grid/Flex, Custom Properties, SVG), ES6+ JavaScript
- **Визуализация**: `Chart.js` (подгружается через CDN)
- **Архитектура**: Single File Component (вся логика, стили и разметка в `index.html`)
- **Данные**: Встроенный JSON-массив `DB` (имитация локального API/моков)

## 📥 Установка и запуск
Приложение не требует сборки, Node.js или локального сервера.
